LIBROS Hackathon: Digitalna Demokracija na
Solana Mreži
Dobrodošli na LIBROS Hackathon! Danas niste samo programeri; vi ste arhitekti povjerenja u
svijetu gdje se glasovi prečesto gube, mijenjaju ili ostaju skriveni u netransparentnim bazama
podataka.
Zamislite školski hodnik ili učionicu. Treba donijeti odluku: Kamo idemo na izlet? Koju boju
majica biramo za kraj godine? Tko će nas predstavljati kao predsjednik razreda? U tradicionalnom
svijetu, to su papirići u kutiji ili podignute ruke koje netko može krivo prebrojati. U našem svijetu,
to su podaci koji moraju biti nepromjenjivi, javno provjerljivi i apsolutno sigurni.
Vaš zadatak je izgraditi decentraliziranu aplikaciju koja će stati na kraj sumnjama. Koristeći
snagu Solana blockchaina i preciznost Anchor frameworka, vaša misija je stvoriti sustav u kojem:
• Identitet je ključ: Svaki korisnik pristupa glasanju putem svog Phantom walleta. Nema
lažnih profila, samo kriptografski potvrđene adrese.
• Transparentnost je zakon: Svako kreirano pitanje i svaki dani glas zapisuju se trajno na
Devnet mrežu. Rezultati nisu pohranjeni na nečijem privatnom serveru, već su javno
dostupni svima na uvid.
• Integritet je neupitan: Jednom kada je glas predan, on je uklesan u digitalni kamen. Vaš
smart contract mora osigurati da nitko – pa čak ni administrator – ne može promijeniti
rezultat ili glasati dva puta.
Projektni zadatak: Blockchain aplikacija za glasanje
koristeći Solana i Anchor
Opis zadatka
Cilj projekta je razviti decentraliziranu web aplikaciju za glasanje koristeći:
● Solana blockchain (Devnet)
● Anchor framework za razvoj smart contracta u Rustu
● Web frontend (React / Next.js)
● Phantom wallet za autentifikaciju korisnika
Aplikacija omogućava učenicima kreiranje anketa i glasanje za određene opcije (npr. izlet, film,
majica razreda, predsjednik razreda itd.).
Svi glasovi moraju biti zapisani na Solana blockchainu.
Funkcionalni zahtjevi
Obavezne funkcionalnosti
1. Spajanje walleta
Aplikacija mora omogućiti:
● spajanje Phantom walleta
● prikaz povezane wallet adrese
● disconnect wallet opciju
2. Kreiranje glasanja
Korisnik mora moći:
● kreirati novo glasanje
● unijeti naslov pitanja
● definirati više opcija za glasanje
Primjer:
Pitanje:
“Gdje idemo na izlet?”
Opcije:
● Zadar
● Split
● Varaždin
3. Glasanje na blockchainu
Glasanje mora koristiti Solana smart contract.
Potrebno je implementirati:
● spremanje glasova na blockchain
● pozivanje Anchor instrukcija
● verifikaciju da wallet može glasati samo jednom
● zabranu izmjene glasa
● zabranu administratorskog mijenjanja glasova
4. Prikaz rezultata
Aplikacija mora prikazivati:
● broj glasova po opciji
● ukupni broj glasova
● tko je glasao (wallet adrese)
5. Pohrana podataka
Svi podaci moraju biti spremljeni na Solana Devnet mreži:
● glasanja
● opcije
● wallet adrese glasača
● broj glasova

1. Instalacija razvojne okoline
Korak 1 — Instalacija WSL2
Windows PowerShell (Administrator)
wsl --install
Restart računala.
Korak 2 — Instalacija Ubuntu distribucije
Instalirati Ubuntu iz Microsoft Store-a.
Preporučena verzija:
● Ubuntu 22.04 LTS
Nakon instalacije:
sudo apt update && sudo apt upgrade -y
2. Instalacija Node.js
Instalacija NVM-a
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash
Restart terminala.
Instalacija Node.js
nvm install --lts
nvm use --lts
Provjera:
node -v
npm -v
3. Instalacija Rusta
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
Dodati Rust u PATH:
source "$HOME/.cargo/env"
Provjera:
rustc --version
cargo --version
4. Instalacija Solana CLI alata
sh -c "$(curl -sSfL https://release.anza.xyz/stable/install)"
Dodati Solanu u PATH:
export PATH="$HOME/.local/share/solana/install/active_release/bin:$PATH"
Provjera:
solana --version
5. Konfiguracija Solana Devnet mreže
Postavljanje Devneta
solana config set --url https://api.devnet.solana.com
Provjera:
solana config get
Generiranje walleta
solana-keygen new
Provjera adrese:
solana address
Dohvaćanje Devnet SOL tokena
solana airdrop 2
Provjera stanja:
solana balance
6. Instalacija Anchor Frameworka
Instalacija AVM-a (Anchor Version Manager)
cargo install --git https://github.com/coral-xyz/anchor avm --locked --force
Restart terminala.
Instalacija Anchor verzije
avm install latest
avm use latest
Provjera:
anchor --version
7. Instalacija Yarn-a
npm install --global yarn
Provjera:
yarn -v
8. Kreiranje Anchor projekta
anchor init voting-dapp
cd voting-dapp
Struktura projekta:
programs/ -> Rust smart contract
app/ -> frontend aplikacija
tests/ -> testovi
Anchor.toml -> Anchor konfiguracija
9. Pokretanje lokalnog validatora
U jednom terminalu:
solana-test-validator
10. Build i deploy smart contracta
Build
anchor build
Deploy
anchor deploy
11. Smart contract funkcionalnosti
